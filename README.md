# steal-instagram
Social reference extractor (Claude/Paperclip skill, Kakashi Phase-5 specialized). SKILL.md at root.

Two modes, three platforms (Instagram · YouTube · TikTok):
- **POST mode** — one post/reel/short → on-screen copy, design tokens, hook structure, funnel → structured reference.
- **ACCOUNT mode** — a profile/channel → content list (with engagement) → hook-formula/format/funnel analysis → **Keens application plan + kr_batch.json hints**.

Requires a browser logged in to the target platform. Extracts *patterns*, generates *original* work (see Ethics in SKILL.md). See `SKILL.md`.
