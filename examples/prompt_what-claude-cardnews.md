## Stealable patterns (우리에게 직접 적용)

### 1. 카드 본문 구조 스켈레톤 (검증된 한국어 listicle)
캡션이 밝힌 자동 생성 구조: **도입 → 문제 정의 → 관점 전환 → 체크리스트 → CTA**.
- 우리 현재 구조(cover → problem/insight → cta)에 **‘관점 전환(perspective shift)’ 과 ‘체크리스트(checklist)’ 카드 타입**을 추가하면 더 풍부해짐.
- content_engine_prompts.md의 3단계 카피 생성에 이 스켈레톤을 옵션 템플릿으로 넣을 것.

### 2. 댓글 키워드 → DM 리드마그넷 (퍼널 검증)
CTA: 한 단어 키워드를 댓글에 남기면 자료를 DM으로 보내주는 구조 → 댓글 5.9천 유발(좋아요보다 많음!). 댓글창이 그 키워드로 도배됨 = ManyChat 키워드 트리거가 실제로 폭발적 인게이지먼트를 만든다는 증거.
- **우리 `dm_keyword` 설계가 정확히 이 메커니즘.** 다만 카피 패턴을 보강: CTA 카드 + 캡션에 "**[무료 자료]를 받으려면 댓글에 '[키워드]'**" 리드마그넷 공식을 명시.
- Keens 적용: "내 무대 진단 체크리스트 받기 → 댓글 '레벨테스트'" 식으로 가치 제안 + 키워드를 묶기.

### 3. 후크(표지) 공식
표지: **[부정 배지: 코딩 X] + [성과 약속] + [비현실적으로 낮은 노력/시간: 30초]**.
- 패턴화: "No-X" 배지(코딩 X / 디자인 X) + 결과 + 시간/노력 최소화. 스크롤 멈춤 강함.
- Keens 톤에 맞게: "재능 X · 8주 만에 오디션 통과 루틴" 류로 변형 가능(단, 보장성 표현은 우리 가드레일 내에서).

### 4. 한·영 병행 캡션
한국어 캡션 + 영어 번역을 한 게시물에 병기. 우리 KO/EN 전략과 일치 — 캡션 생성 시 동일 적용 고려.

### 5. 메타 신뢰 신호
"tool: Claude Opus 4.8", "edit: @계정" 처럼 **제작 도구·에디터 크레딧**을 캡션 끝에 표기 → 전문성·신뢰 신호. 선택적으로 차용 가능.

## 영상 프레임 분석 (로그인 후 실제 영상 캡처 — 2026-06-20)
처음엔 표지+캡션만 봤지만, 로그인 세션에서 23초 릴스를 타임스탬프별로 캡처해 **영상 내부(화면녹화 튜토리얼)** 를 확인함. 핵심:

**그들의 셋업 = Claude Project "카드뉴스 메이커"** (Opus 4.8 높음). 흐름: 새 프로젝트 → 지침 편집 → 주제만 입력 → 10장 PNG 출력.

**프로젝트 지침(시스템 프롬프트) 요지** (화면에 노출됨):
- "You are a premium Korean carousel design director and visual content..." (프리미엄 한국어 캐러셀 디자인 디렉터 역할)
- 작업 방식: **내부적으로 정교한 레이아웃 구성 후 1080×1350 PNG로 출력, 코드는 결과물에 노출하지 않음**.
- **모든 가시 텍스트는 한국어만.**
- **한글 깨짐 방지: Noto Sans KR / Pretendard 사용.**
- "주제만 입력하면 타깃·톤·디자인은 주제에 맞춰 내가 정한다 → 바로 시작."

**결과물 디자인 시스템** (출력 카드에서 추출):
- 컨셉: "**프리미엄 에디토리얼 리포트 스타일**" — 부(富)를 '돈의 기술'이 아닌 '구조'의 문제로.
- 컬러: 차콜 `#0E1116` · 골드 · 오프화이트 · 크림 `#F3EFE6`. (다크 배경 + 골드 포인트 절제)
- 타이포: Noto Sans KR (Black/Bold/Medium/Regular) — 큰 헤드라인으로 명확한 위계.
- 그래픽 어휘: **큰 인덱스 숫자(04/10)**, 공식 카드(수식형), 골드 틱 라인, 저축률 바, 체크박스, 핵심 슬라이드는 톤 반전.
- 페이저 `NN/10` + "넘겨보기 →", 하단 CTA 오버레이 "**세팅법은 댓글에 '클로드'**"(리드마그넷).
- 슬라이드 구조 예: 1장 도입(후크) → … → 핵심(반전) → CTA. 10장.

**우리에게 직접 쓸 업그레이드:**
1. **지침 패턴 차용:** "레이아웃 내부 구성 → PNG만 출력(코드 비노출)" + "가시 텍스트 한 언어 고정" + "폰트 안전(Noto Sans KR/Pretendard)" 3원칙을 우리 content_engine_prompts.md에 명문화.
2. **프리미엄 에디토리얼 비주얼 옵션:** 차콜+골드+크림 + 큰 인덱스 숫자 + 공식/구조 카드 스타일을 Keens용 대체 테마로 추가(스테이지 사진 테마와 별도 선택지).
